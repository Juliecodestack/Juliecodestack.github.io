How to evaluate a bool expression?

Summary: Do you know a bool expression may not return True or False ?

We know that the boolean value of 0 is False, and the boolean value of 1 is True. What about the boolean value of a negative number, for example, -2 ? What is the result of expression True and 3 ?

Let‚Äôs find out the answer.

Here is a question I came across while I was doing a lab of Berkeley CS61A Course . What would Python print?

>>> positive = 28 

>>> while positive: 

...     print("positive?") 

...     positive -= 3

I thought that after printing ‚Äúpositive?‚Äù 10 times , the variable ‚Äúpositive‚Äù would become negative, so the program would stop. But my answer didn‚Äôt pass the test, so I ran it locally to see what happened. To my surprise, the program didn‚Äôt stop! It‚Äôs an infinite loop. Why?

Because the boolean value of a negative number is True. Only the boolean value of 0 is False, while the boolean value of a positive or negative number is True, so bool(-2) is True.

In Python, the boolean value of False (or 0, or ‚Äò‚Äô, or None) is False. The boolean value of others is True.

Then how to evaluate the bool expression?

A bool expression usually contains logic operators such as and, or, not.

(1)and : The result of False and True is False, so for expression value1 and value2 , only when the first element value1 is True do we need to check the second element value2 . For example, the value of expression True and 0 is 0.

That is, and stops evaluating at the first falsy value. For example:

>>> 0 and 2
0
>>> False and -2
False

If all values evaluate to be True, the last value is returned. So the value of expression True and 3 is 3, while the expression 3 and True returns True.

>>> 2 and -3
-3
>>> -3 and 2
2

(2)or: uses an evaluating order like and. The result of True or False is True, so for expression value1 or value2 , only when the first element value1 is False do we need to check the second element value2 .

That is, or stops evaluating at the first truthy value. For example:

>>> 0 or -1
-1
>>> 2 or -1
2
>>>True or 2
True
>>>0 or False or -3 or 1 / 0
-3

Here the last element 1/0 in the last expression won‚Äôt be evaluated, otherwise it‚Äôll produce an error.

If all values evaluate to be False, the last value is returned.

>>> False or 0
0
>>> 0 or False
False

Here we need to pay attention to one thing, the return value of a bool expression like value1 and value2 is not always True or False. It returns value1 or value2, not the boolean value of value1/value2, so it may be a number.

(3)not: Unlike and and or, not expression returns the opposite boolean value of the element, so the result is either True or False.

>>> not 10
False
