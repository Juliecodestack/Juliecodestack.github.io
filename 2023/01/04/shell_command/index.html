Terminal/Shell Commands Cheatsheet

Wed, Jan 4, 2023

I used to think that terminal/shell commands are for Linux users and it looks hard to memorize lots of commands. Now I find I was wrong. These commands are useful and convenient. For example, when runing a Python program in Terminal/Shell, with a cd command we can easily jump to certain directory. Besides, many commands are acronym of the corresponding words, for example, cd stands for “change directory”, so we can figure out the command by thinking of the operation.

Here I will list some commands I’ve learned. I’ll add more as I learned.

1. Directories (or Folders in Windows)

ls : list all the files in the current directory.

pwd: (present working directory) show the current directory.

cd folder1: change directory to folder1.

Three special signs for directory:

.: current directory

..: parent directory, thus cd .. means changing to the parent directory.

~: home directory, thus cd ~ means changing to the home directory, and we can also type cd for the same result.

mkdir folder_name : make a new directory (folder).

rm -r folder_name : remove the whole directory (folder) ( -r means recursively).

Warning: In UNIX, when you rm a file, it’s gone. There is no Recycle bin or Trash for you to “undo” rm, so think twice before you use it.

2. Files

cat filename.filetype : display the contents of the file on the screen.

touch filename.filetype: create a new empty file.

mv source_path dest_path ：move the file/directory from the source_path to the dest_path.

There are two ways of using mv:

1.mv source_file/directory dest_directory : move the source file/directory to the dest_directory

2.mv source_file dest_file : rename the source_file, now the file name is dest_file. If dest_file has already exist, it will be overwritten as the content of the source file.

cp file/directory dest_directory: copy the file/directory to the dest_directory.

For mv/ cp command, if the dest_directory is the current directory, we can write . instead.

rm file_name: remove a single file. Here only a file is removed, so -r isn’t needed.

unzip file_name.zip : unzip the .zip file.

3. Miscellaneous

echo: similar to print function

echo content : print the content on the screen.

echo content>>filename.filetype: write the content to the file.

man command_name: display manual pages for the command.
